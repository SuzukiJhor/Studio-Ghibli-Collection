# Security Policy

## Supported Versions
Only the latest version of the dashboard is supported with security updates.

## Reporting a Vulnerability
If you discover a security issue, please do not open a public issue. 
Contact me at [jhordan.kendy1@gmail.com].

## Data Privacy
This application is a client-side dashboard:
- It does not store user passwords.
- It only communicates with the Studio Ghibli API.
- All "Watched" or "Favorite" data is stored locally in your browser (LocalStorage).
